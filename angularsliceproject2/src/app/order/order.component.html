mat-select> is a form control for selecting a value from a set of options, similar to the native select> element. You can read more about selects in the Material Design spec. It is designed to work inside of a mat-form-field element. To add options to the select, add mat-option> elements to the mat-select>. Each mat-option has a value property that can be used to set the value that will be selected if the user chooses this option. The content of the mat-option is what will be shown to the user. Angular Material also supports use of the native select> element inside of mat-form-field. The native control has several performance, accessibility, and usability advantages. See the documentation for form-field for more information.
